# Hush Cafe - Full Stack Website

A complete, production-ready website for Hush Cafe (a quiet luxury coffee shop). Built with HTML/CSS/JS frontend and Node.js/Express/SQLite backend.

## Features

- ✅ Responsive design (mobile-friendly)
- ✅ Testimonial slider with auto-rotation
- ✅ Table reservation system
- ✅ Contact form with database storage
- ✅ Admin panel to view submissions
- ✅ Google Maps integration
- ✅ WhatsApp & call buttons

## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript (ES6+)  
**Backend:** Node.js, Express.js  
**Database:** SQLite3  
**Icons:** Font Awesome 6
